История того, как я поставил этот сайт (и буду ставить другие сайты) на хостинг и расскажу как это сделать!

С начала я подумал о github там можно выкладывать проекты. Получается, что они хранятся на сайте, который стоит на хостинге значит можно выложить файлы и поставить их на хостинг.

Для начала в github нужно создать новый репозиторий под названием «username».github.io там, где username должен быть ваш никнем (без кавычек). Далее нужно добавить папку, нажав на кнопку создать файл и в названии файла ввести следующее: «название папки»/ «название файла» тоже без кавычек. Потом нужно добавить файлы сайта в наш репозиторий нажав кнопку upload file. Вот и всё теперь наш сайт стоит на хостинге, осталось ввести нашу ссылку в браузере, ссылка будет такая: «ваш никнем».github.io»/«название папки с сайтом»/«Название файла сайта».«расширение файла(у меня .html например: Сайт.html)».

В итоге у меня получилась такая ссылка:

https://shto-to.github.io/Сайты/Как_поставить сайт на хостинг в github/Сайт.html.
